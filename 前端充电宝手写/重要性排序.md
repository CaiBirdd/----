# 前端实习手写题重要性总排序（实战版）

> 适用对象：**前端实习 / 校招初级**
> 排序原则：**不会 = 很可能直接被刷**

## 🟥 Tier 0：必会生存题（死记硬背也要会）
**出现率极高 + 容错率极低**  
👉 **这一档：写不出来，基本没有后续技术追问**

### JavaScript 基础
1. 手写 **防抖函数 (Debounce)**
2. 手写 **节流函数 (Throttle)**
3. 手写 **call / apply / bind**
4. 手写 **new 操作符**
5. 手写 **instanceof 方法**
6. **实现深拷贝**
7. 手写 **Promise.all**

### 数据处理
1. **实现数组去重**
2. **实现数组扁平化 (Flatten)**
3. **实现 add(1)(2)(3)**
4. **解析 URL Params 为对象**
5. **将数字每千分位用逗号隔开**

### 场景应用
1. **循环打印红黄绿**
2. **每隔 1 秒打印 1,2,3,4**

---

## 🟧 Tier 1：高频必会（决定你是不是“不错的实习生”）
**非常爱考，会了明显拉开差距**  
👉 **这一档：开始考“你是不是真的理解 JS”**

### JavaScript 基础
1. **手写 Promise.race**
2. **手写类型判断函数**
3. **函数柯里化的实现**
4. **手写 Object.create**

### 数据处理
1. **实现数组的乱序输出 (洗牌算法)**
2. **使用 ES5 / ES6 求函数参数的和**
3. **实现日期格式化函数**
4. **将 JS 对象转化为树形结构**

### 场景应用
1. **使用 setTimeout 实现 setInterval**
2. **判断对象是否存在循环引用**
3. **发布-订阅模式**
---

## 🟨 Tier 2：中频进阶题（加分项）
**不一定必考，但写出来很加分**  
👉 **这一档：看你的思维和工程感，不追求完美实现**

### JavaScript 基础
1. **实现 AJAX 请求**
2. **手写 Promise.then**
3. **手写 Promise (核心逻辑即可)**

### 数据处理
1. **使用 reduce 求和**
2. **实现数组元素求和**


### 场景应用
1. **实现 jsonp**
2. **实现斐波那契数列**

---

## 🟩 Tier 3：算法 / 思路型（了解即可）
**出现频率较低，更偏算法或通用逻辑**  
👉 **不会一般不挂，但会了会被夸一句**

### 数据处理
1. **实现非负大整数相加**
2. **实现非负大整数相乘**

### 场景应用
1. **小孩报数问题 (约瑟夫环)**
2. **查找文章中出现频率最高的单词**

---

## 🟦 Tier 4：工程思想题（偏认知，不强求手写）
**更多是“你有没有概念”**  
👉 **实习面试更常是“讲思路”，而不是完整手写**

### 场景 / 设计
1. **实现双向数据绑定**

---

## 🧠 一句话使用说明（很重要）
- **时间紧（80%的人）** 👉 只刷 **Tier 0 + Tier 1**
- **时间正常** 👉 再补 **Tier 2**
- **想冲好公司** 👉 挑几道 **Tier 3 / 4** 会讲即可

> **前端实习手写题不是比“广度”，而是比：你会不会那十几道“面试官闭着眼都能问出来的题”。**
> 你这 38 道题，**Tier 0 + Tier 1 已经覆盖了 90% 实习面试现场。**
